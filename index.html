Lesson 2 | Load a map into a web app

Starters:

  • Stage 1 Starter — Basic
    Load a web map from ArcGIS Online or Portal
    Search ArcGIS Online for web maps
  • Stage 2 Starter — Basic
    Load a web scene from ArcGIS Online or Portal
    Search ArcGIS Online for web scenes
  • Stage 3 Starter — Basic
    Modify the Get started with MapView - Create a 2D map sample so it loads a web scene
  • Challenges:
    1. Using your Stage 1 solution, program the buttons to change the basemap used in the webmap
    2. Present both a 2D web map and 3D web scene side-by-side

Example Solutions:

  • Stage 1 - Load a WebMap
  • Stage 2 - Load a Web Scene
  • Stage 3 - Convert a sample to display a web scene
  • Challenge 1
  • Challenge 2

Additional Solutions:

  • Stage 1 — Load a web map
  • Stage 2 — Load a web scene

Workflow

  1. Determine webmap ID (PortalItem ID)
  2. Load required modules
  3. Create instance of WebMap
  4. Set WebMap's portalItem property to PortalItem ID
  5. Create instance of MapView
  6. Set MapView's map property to WebMap
